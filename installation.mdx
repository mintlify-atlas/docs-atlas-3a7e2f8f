---

CloudStream is not available on the Google Play Store. You install it by downloading the APK directly from the official GitHub repository and sideloading it onto your device.

<Info>
  CloudStream requires **Android 6.0 (API level 23)** or higher.
</Info>

<Warning>
  Only download CloudStream from the official repository at [github.com/recloudstream/cloudstream](https://github.com/recloudstream/cloudstream). APKs distributed elsewhere may be modified or malicious.
</Warning>

## Install from GitHub Releases

<Steps>
  <Step title="Enable installation from unknown sources">
    Android blocks app installs from outside the Play Store by default. To allow it:

    1. Open **Settings** on your device.
    2. Go to **Apps** (or **Apps & notifications**) → **Special app access** → **Install unknown apps**.
    3. Select the browser or file manager you will use to open the APK.
    4. Toggle **Allow from this source** on.

    The exact path varies by Android version and device manufacturer. On some devices the option is under **Security** → **Unknown sources**.
  </Step>

  <Step title="Download the APK from GitHub Releases">
    Open your browser and go to the [CloudStream releases page](https://github.com/recloudstream/cloudstream/releases).

    You will see two types of builds:

    - **Stable** — recommended for most users. These releases are tagged with a version number (e.g., `v4.7.0`) and have been through more testing.
    - **Pre-release** — built more frequently from the latest code. These include the newest features and fixes but may be less stable. Pre-release APK filenames end in `-PRE`.

    Download the APK file for the build you want.
  </Step>

  <Step title="Install the APK">
    Open the downloaded APK file using your file manager or directly from your browser's download notification.

    Tap **Install** when prompted. Android may show a security warning — this is expected for any app installed outside the Play Store.

    Once installation completes, tap **Open** to launch CloudStream, or find it in your app drawer.
  </Step>
</Steps>

## Install from F-Droid

CloudStream is available through the [IzzyOnDroid F-Droid repository](https://apt.izzysoft.de/fdroid/index/apk/com.lagradost.cloudstream3). If you use F-Droid, add the IzzyOnDroid repo and search for **CloudStream**.

<Note>
  F-Droid builds may lag behind the latest GitHub release. For the most up-to-date version, use the GitHub Releases method above.
</Note>

## Updating CloudStream

CloudStream can check for and apply updates from within the app. Go to **Settings** → **General** to check for updates manually. The app will download and install the new APK automatically when an update is available.
